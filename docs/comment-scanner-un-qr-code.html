Comment scanner un QR code ?

Réponse courte

Ouvrez simplement l'appareil photo de votre téléphone et visez le code : aucune application n'est nécessaire sur les téléphones récents.

1

Ouvrez l'appareil photo

iPhone comme Android : cadrez le QR code avec l'appareil photo natif. Une notification apparaît en haut de l'écran — touchez-la.

2

Rien ne s'affiche ? Activez l'option

Android : dans les réglages de l'appareil photo, activez « Scanner les QR codes ». Sinon, Google Lens fait le travail.

3

Vérifiez l'URL avant d'ouvrir

Lisez le lien proposé dans la notification : le domaine juste avant le premier « / » doit correspondre au site attendu. Au moindre doute, ne l'ouvrez pas.

4

Méfiez-vous des QR codes collés

Sur les parkings ou les bornes, des fraudeurs collent leurs propres codes par-dessus les originaux (« quishing »). Un autocollant récent est un signal d'alerte.

Scanner est devenu un geste natif — le vrai réflexe à acquérir est la vérification du lien avant d'ouvrir. Un doute sur un QR code précis ? Décrivez-le à l'Expert.
