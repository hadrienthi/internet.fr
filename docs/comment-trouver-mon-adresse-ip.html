Comment trouver mon adresse IP ?

Réponse courte

Votre IP publique s'affiche en un clic — elle est déjà visible dans le test de débit d'Internet.fr. L'IP locale, elle, se trouve dans les paramètres réseau de votre appareil.

1

IP publique : celle que voit Internet

Attribuée par votre opérateur, c'est l'adresse vue par les sites web. Elle s'affiche directement dans le widget Test de débit d'Internet.fr.

2

IP locale sur ordinateur

Windows : Paramètres → Réseau et Internet → Propriétés. Mac : Réglages → Wi-Fi → Détails. Elle commence le plus souvent par 192.168.

3

IP locale sur mobile

iPhone comme Android : Réglages → Wi-Fi → touchez le réseau auquel vous êtes connecté. L'adresse IP apparaît dans les détails.

4

IPv4 ou IPv6 : les deux sont normales

92.174.13.58 est une IPv4 ; une forme longue avec des « : » est une IPv6. Les deux formats coexistent et fonctionnent en parallèle.

Publique pour le monde extérieur, locale pour votre réseau domestique : ce sont deux adresses différentes et c'est normal. Dites à l'Expert ce que vous cherchez à faire (ouvrir un port, configurer une caméra…) et il vous guide.
